###安裝方式

##### 本次作品使用 [XAMPP](https://www.apachefriends.org/zh_tw/download.html) 安裝模組來開發

##### 解壓縮後將整個資料夾放入此模組的 Web Server 的根目錄下

Ex: 你的硬碟:\xampp\htdocs
